## C&C Mental Omega

### [Installing](http://mentalomega.com/index.php?page=install)
1. Prepare files from Red Alert 2 Yuri's Revenge 1.001

The files required for Mental Omega 3.3 to function are:

- BINKW32.dll
- BLOWFISH.dll
- gamemd.exe (patched to version 1.001)
- ra2.mix
- ra2md.mix
- language.mix
- langmd.mix
